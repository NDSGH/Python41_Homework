1. Запускаем тестовый сервер. Переходим по ссылке (автоматически попадаем на окно с формой регистрации).

2. Регистрируемся: вводим логин (имя пользователя) и пароль (автоматически попадаем на оно с формой входа).

3. Авторизуемся: вводим ранее введённые логин (имя пользователя) и пароль (если пользователь зарегистрирован, то попадаем на окно с приветствием).

4. Выход из учётной записи пользователя производит при помощи кнопки "Logout".
