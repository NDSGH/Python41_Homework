Исправьте программу tcp.py так, чтобы к ней можно было обратиться открыв страницу в браузере по адресу http://127.0.0.1:7001/ - браузер должен отобразить HTML страницу с произвольным сообщением.
То есть:
 - оставьте только серверную часть
 - изменив протокол сообщений который в программе подразумевает передачу 4 байт длины тела сообщения, следом за которыми идет тело сообщения на минимальную реализацию текстового протокола HTTP
 - Минимальная реализация протокола HTTP подразумевает что на любой запрос HTTP GET от браузера программа должна отправить валидный HTTP ответ для браузера с HTML страницей внутри которой будет ваш текст.
 - Посмотреть формат текстового протокола HTTP можно на Mozilla Developer Network, в любых других источниках, либо увидеть в программе Wireshark
